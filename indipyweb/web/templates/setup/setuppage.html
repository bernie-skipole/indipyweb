## Create backup button

As users are created, they will be set into the database 'indipyweb.db'. Besides user data the following values will be read after the service is restarted.

The folder containing the database should be set on the command line argument, if not set the current working directory will be used.

To move the database file, create a backup using the above button, and move and rename the backup file.

If web host and port are set on the command line, they will have priority, if not set, then the values from the database will be used.
